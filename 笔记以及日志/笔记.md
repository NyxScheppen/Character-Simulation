## P2 当我们在聊数据库的时候，我们在聊什么
- 
### 基础概念
- 字段：表里的“列”，表示这类数据有哪些属性
- 记录：表里的一行，表示一条具体数据
- 主键：用来唯一标识一条记录的字段，不能和其他记录重复
- 外键：一张表里用来指向另一张表主键的字段
- 字段类型
    - Integer
    - String
    - Text
    - Boolean
    - DateTime
- session：和数据库进行一次“操作会话”的对象。 

### SQLAlchemy
- Python 里用来操作数据库的工具
#### 常用函数
- `db.add(obj)`：把对象加入当前会话，准备写入数据库
- `db.commit()`：正式提交改动到数据库
- `db.refresh(obj)`：从数据库刷新对象，拿到最新状态
- `db.query(Model)`：开始查询某个模型对应的表
- `.all()`：取出所有结果
- `.first()`：取出第一条结果
- `.filter(...)`：给查询添加条件
- `db.delete(obj)`：删除对象
- `db.rollback()`：提交失败时回滚改动
- `db.close()`：关闭数据库会话

app/core/database.py
```python
from sqlalchemy import create_engine
from sqlalchemy.orm import declarative_base, sessionmaker

# 表示数据库使用 SQLite，数据库文件叫 app.db
DATABASE_URL = "sqlite:///./app.db"

# engine是数据库的链接核心对象
engine = create_engine(
    DATABASE_URL,
    connect_args={"check_same_thread": False}
)
# 一个可调用的对象，用来生成数据库 Session
SessionLocal = sessionmaker(
    autocommit=False,
    autoflush=False,
    bind=engine
)
# 所有model都继承这个Base
Base = declarative_base()

# 每次请求来时，开一个数据库 Session，请求用完后自动关闭
def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
```

```py
from sqlalchemy.orm import Session
from app.models.character import Character

def get_by_id(db: Session, character_id: int) -> Character | None:
    return db.query(Character).filter(Character.id == character_id).first()

def get_by_name(db: Session, name: str) -> Character | None:
    return db.query(Character).filter(Character.name == name).first()

def get_all(db: Session) -> list[Character]:
    return db.query(Character).all()

def create(db: Session, name: str, description: str | None = None) -> Character:
    character = Character(
        name=name,
        description=description
    )
    db.add(character)
    db.commit()
    db.refresh(character)
    return character

def delete(db: Session, character: Character) -> None:
    db.delete(character)
    db.commit()
```

app/services/character_service.py
```py
from sqlalchemy.orm import Session
from app.models.character import Character
from app.schemas.character import CharacterCreate

# 由 api 层通过依赖注入拿到 db，再传给 service
def create_character(db: Session, payload: CharacterCreate) -> Character:
    name = payload.name.strip()
    description = payload.description.strip() if payload.description else None

    if not name:
        raise HTTPException(status_code=400, detail="角色名不能为空")

    existing_character = character_repository.get_by_name(db, name)
    if existing_character:
        raise HTTPException(status_code=400, detail="角色名已存在")

    return character_repository.create(db, name=name, description=description)

def get_all_characters(db: Session) -> list[Character]:
    return character_repository.get_all(db)

def get_character_by_id(db: Session, character_id: int) -> Character:
    character = character_repository.get_by_id(db, character_id)
    if not character:
        raise HTTPException(status_code=404, detail="角色不存在")
    return character
// dict为返回类型注释
def delete_character_by_id(db: Session, character_id: int) -> dict:
    character = character_repository.get_by_id(db, character_id)
    if not character:
        raise HTTPException(status_code=404, detail="角色不存在")

    character_repository.delete(db, character)
    return {"message": "删除成功"}
```

```py
    return get_all_characters(db)
```

## PDay3 那么，前端呢？
### 基础概念
- 组件：页面里的一个可复用的模块，例如顶部导航栏
- props：父组件传给子组件的数据，本质上是给组件传参数
- state：组件自己管理的数据
- useState：React 提供的“保存可变化数据”的工具。
- useEffect：组件显示出来以后，要顺便做一些事
- axios：前端发 HTTP 请求的工具
- JSX：可以在 JavaScript 里写“像 HTML 一样的界面结构”
- Vite：前端开发工具，创建、运行、打包 React 项目。
- CORS / 跨域：前端页面的来源和后端接口的来源不一样

#### 前端请求后端的基本流程
```sh
浏览器
  ↓
访问 Vite 提供的前端页面（localhost:5173）
  ↓
React 页面运行起来
  ↓
axios 请求 FastAPI（127.0.0.1:8000）
  ↓
FastAPI 返回 JSON
  ↓
React 把数据渲染到页面
```

src/components/CharacterList.jsx
```jsx
import { useEffect, useState } from "react";
import axios from "axios";
import CharacterCard from "./CharacterCard";

function CharacterList() { 
  const [characters, setCharacters] = useState([]); // 角色列表
  const [loading, setLoading] = useState(true);// 是否在加载
  const [error, setError] = useState("");// 错误信息，初始时空字符串
// 页面加载后请求接口
  useEffect(() => {
    // res是后端返回的JSON
    axios
      .get("http://127.0.0.1:8000/api/v1/characters")
      .then((res) => {
        // 重新加载页面
        setCharacters(res.data);
      })
      .catch((err) => {
        console.error("获取角色失败", err);
        setError("获取角色失败，请检查后端是否启动");
      })
      // 无论成功与否都结束loading
      .finally(() => {
        setLoading(false);
      });
  }, []);// []表示这个effect“页面一加载就请求一次”

  if (loading) {
    return <p>加载中...</p>;
  }

  if (error) {
    return <p>{error}</p>;
  }
    // 渲染角色列表
  return (
    <div>
      <h2>角色列表</h2>
      {characters.length === 0 ? (
        <p>暂无角色</p>
      ) : (
        // 把characters 数组里的每一项都变成一个 CharacterCard
        characters.map((item) => (
          <CharacterCard
            key={item.id}
            name={item.name}
            description={item.description}
          />
        ))
      )}
    </div>
  );
}

export default CharacterList;
```
